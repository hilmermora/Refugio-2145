# Refugio-2145
Ano 2045. Um evento chamado “A falha” dizimou grande parte das cidades. Criaturas mutantes — chamadas genericamente de Vácua — dominam vastas áreas.  Comunidades remanescentes criam portais informativos para ensinar aos sobreviventes a reconhecerem cada criatura, descobrir pontos fracos, táticas de sobrevivência e locais seguros.
